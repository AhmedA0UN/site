Heap & Priority Queue
الكومة هي شجرة ثنائية كاملة تضمن أن الأب دائماً أكبر (Max-Heap) أو أصغر (Min-Heap) من أبنائه. تُستخدم في Heap Sort وأسئلة Top-K الشهيرة.

10.1 ما هي الكومة؟

💡 التعريف

10.2 Priority Queue في C++

C++ — priority_queueheap_usage.cpp
#include <queue>
using namespace std;

// ── Max-Heap (الافتراضي) ───────────────────────
priority_queue<int> maxH;
maxH.push(30); maxH.push(10); maxH.push(50);
maxH.top(); // 50 — الأكبر دائماً

// ── Min-Heap ───────────────────────────────────
priority_queue<int, vector<int>, greater<int>> minH;
minH.push(30); minH.push(10); minH.push(50);
minH.top(); // 10 — الأصغر دائماً

// جميع العمليات: Push/Pop O(log n) | Top O(1)

10.3 تمارين LeetCode Style

مسألة 1: Kth Largest Element in ArrayMedium AmazonGoogleMeta

المشكلة: أوجد العنصر الـ k الأكبر في المصفوفة.

مثال: [3,2,1,5,6,4], k=2 → 5

💡استخدم Min-Heap بحجم k. إذا تجاوز الحجم k، احذف أصغر عنصر. الناتج: top() هو الـ k الأكبر.
C++ — Min-Heap O(n log k)
int findKthLargest(vector<int>& nums, int k) {
    priority_queue<int, vector<int>, greater<int>> minH;

    for (int n : nums) {
        minH.push(n);
        if (minH.size() > k) minH.pop(); // احذف الأصغر
    }
    return minH.top(); // الـ k الأكبر
}
// Time: O(n log k) | Space: O(k)
مسألة 2: Top K Frequent ElementsMedium AmazonGoogleMetaMicrosoft

المشكلة: أوجد الـ k عناصر الأكثر تكراراً.

مثال: [1,1,1,2,2,3], k=2 → [1,2]

C++ — HashMap + Min-Heap O(n log k)
vector<int> topKFrequent(vector<int>& nums, int k) {
    unordered_map<int,int> freq;
    for (int n : nums) freq[n]++;

    // Min-Heap: {frequency, value}
    priority_queue<pair<int,int>,
        vector<pair<int,int>>,
        greater<>> minH;

    for (auto& [val, cnt] : freq) {
        minH.push({cnt, val});
        if (minH.size() > k) minH.pop();
    }

    vector<int> res;
    while (!minH.empty()) {
        res.push_back(minH.top().second);
        minH.pop();
    }
    return res;
}
